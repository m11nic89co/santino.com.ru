Carousel images guidelines

- Aspect ratio: 1:1 (square)
- Recommended size (2x): 640x640 px (min). Better: 960x960 px if available.
- Format: WebP (preferred) or JPEG for photos; PNG/WebP lossless for sharp graphics.
- Safe area: keep important content centered (object-fit: cover crops edges).
- Target weight: ~80–150 KB per image.
- File naming: slide-01.webp, slide-02.webp, ... (use lowercase, hyphens).
- Placement: put files into this folder and update `section1-carousel.js` image sources accordingly.
