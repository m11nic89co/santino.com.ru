# Банк SVG‑чертежей (blueprints)

Этот каталог хранит **профессиональные SVG‑чертежи**, которые используются как фоновые элементы
для подчёркивания технологической направленности сайта.

## Структура

- `hero-*.svg` — чертежи, предназначенные в первую очередь для hero‑секции.
- `contract-*.svg` — чертежи для секции «Контрактное литьё».
- `generic-*.svg` — универсальные чертежи, которые можно использовать в любых секциях.

Рекомендуемая цель — **не менее 20 файлов**:

- `hero-01.svg` … `hero-08.svg`
- `contract-01.svg` … `contract-06.svg`
- `generic-01.svg` … `generic-06.svg`

## Требования к SVG

- Чистый вектор без растра, без лишних `<style>`/`<script>`.
- Цвета:
  - контуры: оттенки светло‑голубого/стального (например `#f4f7ff`, `#8ea0c7`);
  - фон: прозрачный (фон задаётся через CSS/контейнер).
- Размеры:
  - `viewBox` вида `0 0 420 260` или кратные этой пропорции;
  - без жёстких `width/height` в px, чтобы SVG хорошо масштабировался.

## Процесс утверждения

1. Чертежи рисуются и сохраняются в этот каталог по схемам имён выше.
2. Перед подключением в продакшен мы просматриваем их визуально и при необходимости дорабатываем.
3. После утверждения конкретные файлы будут подключены в модуле
   `assets/js/modules/ui/animations/blueprint-bank.config.js` и далее использованы в
   `blueprint-background`.
